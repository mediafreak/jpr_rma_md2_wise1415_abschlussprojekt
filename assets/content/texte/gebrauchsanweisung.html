Erfinden Sie eine Gebrauchsanweisung für einen beliebigen Gegenstand oder eine beliebige Handlung. Sie dürfen (müssen aber nicht) eigene Bilder und Fotos benutzen.

Gliedern Sie Ihre Gebrauchsanweisung in mindestens drei bis maximal fünf Schritte. Benutzen Sie pro Schritt mindestens fünf Zeilen erklärenden Text.

Die Gebrauchsanweisung soll interaktiv funktionieren, als ob man im Web nachsehen muss, wie die Handlung ausgeführt werden soll oder wie das Produkt bedient werden soll.

Geben Sie Ihren Schritten Überschriften und nummerieren Sie sie deutlich. Sie könnten Überschriften oder Zahlen animieren oder als mouseover definieren.
