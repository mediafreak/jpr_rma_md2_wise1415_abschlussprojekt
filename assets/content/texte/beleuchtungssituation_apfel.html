Nehmen Sie eine Frucht oder Gemüse, platzieren Sie das Objekt auf einen neutralen Hintergrund und machen Sie eine Reihe von Aufnahmen mit mindestens fünf verschiedenen Beleuchtungssituationen.

Verwenden Sie für die ersten drei nur eine einzige Lichtquelle. Versuchen Sie bei den beiden anderen Aufnahmen, das Objekt so auszuleuchten, dass es z. B. keine Schatten hat oder es besonders dramatisch aussieht (Schatten!).

Wirkung von Farbfiltern auf einem Apfel (Entwurf)

Der Apfel zeigt die Anwendung von Farbfiltern. Interessante Strukturen können hierbei zum Vorschein kommen oder durch Auslöschung verschwinden.
