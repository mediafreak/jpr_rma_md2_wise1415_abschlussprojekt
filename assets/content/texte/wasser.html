Fotografieren Sie Wasser in Bewegung. Versuchen Sie zuerst, das Wasser in seiner Bewegung einzufrieren, indem Sie eine sehr kurze Verschlusszeit wählen. Achten Sie darauf, dass Sie genügend Licht zur Verfügung haben, sonst können Sie keine kurzen Zeiten wählen. In einem zweiten Versuch lassen Sie den Verschluss länger offen, damit das Wasser verschwimmt. Benutzen Sie ein Stativ, sonst verwackelt die Aufnahme..
